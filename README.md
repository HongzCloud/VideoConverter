## 앱 설명
동영상을 오디오로 간편하게 변환하는 앱  

주요기능 
- 동영상 -> 변환(wav, mp3, m4a, caf)
- 동영상/오디오 재생
- 동영상/오디오 공유하기
- 백그라운드 재생
